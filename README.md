# Hungry-food

it is a food ordering app using React and Swiggy's API.

## Tech Stack

- React
- Parcel
- Tailwind CSS
- Jest(Testing)
- Redux-toolkit

## Features

- Get restaurants by current location
- Add to cart
- Shimmer UI
-

## How to use

1. Clone the repo

```bash
git clone https://github.com/DineshRout779/FoodHunt
```

2. Install dependencies

```bash
npm install
```

3. Start the dev server

```bash
npm start
```
